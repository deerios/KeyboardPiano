# Keyboard Piano

This project currently only supports DrunkDeer keyboards. If you'd like to add support for your magnetic keyboard, please create a pull request.

In addition, this project also supports the use of video game steering wheel pedals to control the sustain (brake pedal) and some other user-configurable modes for the clutch pedal, such as:

- Feather Mode:
  - When activated, moves the lower-velocity envelope further into the moderate key-press range for even more fine control over soft notes.
- Octave Double Mode
  - When activated, creates an effect that thickens a sound by adding one or more detuned and pitch-shifted voices an octave above and/or below the original signal. It blends the original tone with shifted duplicates to create a richer, fuller, and more harmonically complex sound
- Transpose Mode
  - When activated, applies a constant musical interval (semitones) to every note.
